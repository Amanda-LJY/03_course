1.shell指令
  Linux中文件夹也叫作目录
  1)pwd,cd,ls
    pwd:print working directory
      打印当前所在的路径
    ls:list
      显示当前目录中的所有非隐藏文件或目录
      蓝色(后缀/)：表示文件夹(目录)
      白色：表示文件
      绿色：后缀(*):表示可执行文件(双击可运行的)
            后缀(lnk):超链接，快捷方式    
          ~/Desktop ~：/c/Users/Administrator 主目录
      ls -a 显示所有的文件或目录,包括隐藏的
          Linux系统中,文件的隐藏式通过文件名前面加.实现的
    cd:change directory
      切换工作目录
      cd a 切换当期目录下的a目录中去
      cd .. 切换到上一级目录
      cd a/b/c/d
      cd ~：~当前登录用户的主目录
            该指令切换到主目录
  2)mkdir make directory 新建文件夹/目录
        mkdir 目录路径
        mkdir a  在当前目录下新建文件夹a
                  当a存在时，使用该指令会报错
        mkdir a/b 在a的目录下创建b
                  如果父级目录不存在，报错
        mkdir -p a/b 在a的目录下创建b，如果a目录不存在，先创建a，在创建b
  3)touch 新建文件
        touch 文件路径
        touch 1.txt 在当前路径下新建文件1.txt
              如果touch的文件已经存在，则不会创建新的文件，而是修改该文件的最后访问时间
              如果touch的是一个文件夹，则修改文件夹的最后访问时间
  4)rmdir  (只能删除空的目录)
    remove directory 删除文件夹、目录
    rmdir 文件夹路径
    rmdir ./a/b/c 删除a下的b下的c文件夹，只删除c文件夹
  5)rm
    remove 删除文件或非空目录
    rm 文件路径  删除指定文件的路径
    rm -rf 目录的路径 删除非空目录
  6)vi
    文本编辑器
    vi 文件名路径  打开某个文件里输入数据
    进入vi界面后，首先进入的是命令行模式，此时不能输入数据
                输入a/i可以进入输入模式，此时可以输入数据
                输入完成按Esc键退出输入模式，
    
  7)cat
    catch 读取文件内容
    catch 文件名路径 
2.Git的使用
  所有的git命令都是以git开头
  1)配置个人用户信息(只需要配置一次)
    git config --global user.name "你的用户名"
    git config --global user.email "你的邮箱"
  2)初始化本地仓库
    先进入目标文件夹
    使用指令：git init
      此时在目标文件夹里会生成一个隐藏文件夹.git
      同时，命令行路径，会出现master
  3)查看本地仓库的状态
    git status：3个区域4个状态
  4)提交
    1.提交到暂存区
      git add 文件名
    2.将暂存区的内容提交到master
      git commit -m "提交信息"
